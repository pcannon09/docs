\chapter{CIOF -\/ Custom Input Output Format }
\hypertarget{index}{}\label{index}\index{CIOF -\/ Custom Input Output Format@{CIOF -\/ Custom Input Output Format}}

{\ttfamily CIOF} is a lightweight C++17+ input/output framework designed to simplify formatted printing, logging, and user interaction. It provides a set of high-\/level utilities to handle standard output, error logging, ANSI styling, and user input consistently across platforms\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Unified Output Handling}} Output to {\ttfamily std\+::cout}, {\ttfamily std\+::cerr}, or {\ttfamily std\+::clog} with a simple API.
\item {\bfseries{Configurable Output}} Control spacing, delimiters, and formatting globally via {\ttfamily \doxylink{namespaceciof_a835007e60a4807051d32c3352e867119}{ciof\+::output\+Conf}}.
\item {\bfseries{Print \& Echo Helpers}}
\begin{DoxyItemize}
\item {\ttfamily print()} for output with newline
\item {\ttfamily echo()} for output without newline
\item Variants for standard log and error streams
\end{DoxyItemize}
\item {\bfseries{Input Utilities}} Read values interactively with {\ttfamily input()} functions, optionally with a prompt.
\item {\bfseries{ANSI Styling Support}} Easily apply bold, italic, underline, strikethrough, or custom RGB colors.
\item {\bfseries{Cross-\/\+Platform (with limitations)}} Works on Linux, mac\+OS, and Windows 10+. Versions lower than Windows 10 have no support for ANSI
\end{DoxyItemize}

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Configuration}{Configuration}}\label{index_autotoc_md4}
Output behavior can be controlled globally using {\ttfamily \doxylink{structciof_1_1CIOFOutputConfig}{ciof\+::\+CIOFOutput\+Config}}. Options include automatic spacing, custom delimiters, ignoring formatting keywords, and enabling/disabling processing.
