{\ttfamily Var\+Par} (Variable Parser) is a extremely lightweight {\bfseries{C++17+ configuration file parser}} that provides simple, strict key value parsing for flat text configuration files

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Strict Parsing\+:}} Ensures each key is uniquely defined and every line contains a valid {\ttfamily key = value} pair
\item {\bfseries{Automatic Validation\+:}} Checks file existence and verifies the correct file extension before parsing
\item {\bfseries{Informative Error Handling\+:}} Reports precise error messages and line numbers for duplicate or malformed entries
\item {\bfseries{Minimal Dependencies\+:}} Uses only standard C++17 libraries
\item {\bfseries{Lightweight Runtime Status\+:}} Parser state (constructed / error) is tracked through {\ttfamily Parser\+Status} for safe checks
\end{DoxyItemize}

\item Duplicate keys produce an immediate error, halting parsing.
