{\ttfamily \doxylink{structCSTR}{CSTR}} is a lightweight C string utility library that provides a safer and more flexible alternative to raw C-\/style strings. It wraps dynamic allocation, formatting, substring manipulation, and replacement in a consistent API without pulling in heavy dependencies.

All compilers {\bfseries{must support C11}} or higher.\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md5}

\begin{DoxyItemize}
\item {\bfseries{Dynamic String Management}} Create, copy, resize, and destroy strings with automatic memory handling.
\item {\bfseries{Safe Set and Append Operations}} {\ttfamily \doxylink{cstr_8h_a5869f4de55dbc9604a41377a98a5d8d1}{cstr\+\_\+set()}} and {\ttfamily \doxylink{cstr_8h_a7a74e1845b0b12ce83fee782b9106f13}{cstr\+\_\+add()}} handle allocation internally with optional capacity limiting.
\item {\bfseries{Substring and In-\/\+Place Editing}} Includes {\ttfamily \doxylink{cstr_8h_a48a4112d753f753d4d8725fc3e7c5a33}{cstr\+\_\+substr()}}, {\ttfamily \doxylink{cstr_8h_a629281b6e3cecf239e0c1d05cd242b35}{cstr\+\_\+erase()}}, {\ttfamily \doxylink{cstr_8h_a0ba51ccc14c1f17db5b453aa857c92eb}{cstr\+\_\+insert()}}, and {\ttfamily \doxylink{cstr_8h_a3f8f5540a61c0814d0518cd2bad72eb8}{cstr\+\_\+shrink()}}.
\item {\bfseries{Replacement Utilities}} Replace single or all occurrences using {\ttfamily \doxylink{cstr_8h_ad5ffa3660e5c00c93763589cb4d55ea8}{cstr\+\_\+replace()}} and {\ttfamily \doxylink{cstr_8h_a260c899e712785b5e45f07c6b4d6a934}{cstr\+\_\+replace\+All()}}.
\item {\bfseries{Case and Formatting Helpers}} Convert to upper/lower, reverse, clear, or generate strings from numeric values.
\item {\bfseries{Search and Compare Support}} {\ttfamily \doxylink{cstr_8h_ababe5de017b72cf0fbc1c1f74bc22337}{cstr\+\_\+find()}}, {\ttfamily \doxylink{cstr_8h_af211d3b88dcfbfa8d8e467c819017e5d}{cstr\+\_\+find\+From()}}, {\ttfamily \doxylink{cstr_8h_a5d7d68335475e3e92ad7af71c1436f01}{cstr\+\_\+starts\+With()}}, {\ttfamily \doxylink{cstr_8h_afa6b390abaebf90c0ffe07c92662bdcc}{cstr\+\_\+ends\+With()}}, and {\ttfamily \doxylink{cstr_8h_a19f903d2ed994c92fea4c65903c3b552}{cstr\+\_\+comp()}}.
\item {\bfseries{Optional Capacity Locking}} Enabling {\ttfamily force\+Cap} prevents automatic reallocation and returns {\ttfamily CSTR\+\_\+\+FORCECAP\+\_\+\+LIMIT} on overflow attempts.
\end{DoxyItemize}
